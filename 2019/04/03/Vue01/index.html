#Vue基础知识总结(篇1)
Vue 是一套构建用户界面的渐进式框架,也是一种MVVM(Model-View-ViewModel )模式，即数据与视图相互交互的一种设计模式。

一、Vue的安装

1.直接引用（点击 下载 vue 文件，在 html 文件中引入即可。）

2.npm安装（npm install vue）

3.脚手架安装（vue-cli）

公司项目现在一般都是采用的脚手架安装，可用于快速搭建大型单页应用

安装脚手架过程会有相应的提示如下，根据你的需求来选择安装与否。

三、生命周期

每个 Vue 实例在被创建之前都要经过一系列的初始化过程。例如需要设置数据监听、编译模板、挂载实例到 DOM、在数据变化时更新 DOM 等。同时在这个过程中也会运行一些叫做生命周期钩子的函数，给予用户机会在一些特定的场景下添加他们自己的代码。
有兴趣的可以去把整个生命周期研究一遍，这里我主要讲两个经常用到的。

1.created

实例已经创建完成之后被调用,挂载阶段还没开始，暂时还不能进行DOM操作，可以在这个期间进行后台请求。

2.mounted

vm.$el 已挂载在文档内，对已有dom节点的操作可以在这期间进行。
